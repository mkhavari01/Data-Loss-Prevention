# How To Run Project
 
1. git clone https://github.com/mkhavari01/task-cloud.git

2. cd task-cloud

<h4>if you have Docker</h4>
3. docker-compose up  

<h4>if you don't have Docker</h4>
3. cd server uncommnet code line 14 and remove line 13 <br/>
4. npm run start<br/>
5. cd client<br/>
6. npm start
